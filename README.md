# Air_Quality_Index_Project

## Project Overview

Air pollution is a serious environmental issue impacting public health. This project focuses on building a Machine Learning-based AQI prediction system using historical air quality data. Multiple ML algorithms were trained and their performances were compared to identify the best-performing model.

## Objectives

1.Preprocess and clean air quality data

2.Perform exploratory data analysis (EDA)

3.Train and compare multiple machine learning algorithms

4.Evaluate model performance and select the best model

5.Visualize AQI trends and prediction results

## Machine Learning Approach

Data Preprocessing: Handling missing values, cleaning data, and feature selection

Feature Engineering: Identifying key pollutant features influencing AQI

Model Training: Training two different ML algorithms for AQI prediction

Model Comparison: Comparing models using performance metrics

Best Model Selection: Selecting the most accurate and reliable model

Visualization: Interpreting AQI trends and model predictions

## Algorithms Used

Algorithm 1: Random Forest Classifier

Algorithm 2: XGBoost Classifier

## Model Evaluation

Models were evaluated using performance metrics such as:
Accuracy

ROC-AUC

F!-Score

Classification Report 

The best-performing model was selected based on evaluation results and prediction accuracy.

## Technologies Used

Programming Language: Python

Successfully trained and compared two ML models for AQI prediction

Identified the best-performing model based on evaluation metrics
